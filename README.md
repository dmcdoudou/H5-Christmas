# 圣诞许愿活动H5（基于VUE重构）

> 该项目是之前帮某品牌公司做的微信公众号项目，项目上线两周内，吸引了上万用户参与活动。
> 最近学习了ES6和VUE，便基于ES6和VUE对项目进行前端部分的重构，以进一步熟悉ES6和VUE。

## 安装与运行

``` bash
# 安装依赖包
npm install

# 启动热加载服务器： localhost:8080
npm run dev

# 构建项目
npm run build

```
